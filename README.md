# NUMBER SEPARATOR

## Description
This program is an Object-Oriented Graphical User Interface (GUI) tool built with Python and `tkinter`. It reads a text file containing a list of 20 integers, mathematically evaluates them, and separates them into two new output files: one strictly for even numbers and one strictly for odd numbers.

## Features
* **Interactive GUI:** Replaces standard terminal prompts with a clean, interactive window and file explorer dialog.
* **Two-File OOP Architecture:** Strictly adheres to Object-Oriented Programming best practices by separating the class blueprint from the execution script.
* **Automated File Handling:** Reads source data and automatically generates `even.txt` and `odd.txt` in the background.

## How to Use
**1. Prepare the Data:** Create a text file named numbers.txt in the same directory as the scripts. Add 20 random integers to this file, placing one integer on each line.
**2. Run the Program:** Open your terminal or IDE (like VS Code) and execute the main driver file:
  
   `python main_program_for_number_separator.py`
  
**3. Select the File:** When the Graphical Interface pops up, click "Choose File & Separate".

**4. Browse and Execute:** Use the file explorer to locate your numbers.txt file and select it.

**5. View Results:** A success message will appear. Check your folder to see your newly generated even.txt and odd.txt files containing the separated numbers.
